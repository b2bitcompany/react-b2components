# react-b2components

## Documentation

```bash
$ yarn storybook
```

## Structure Folders

- src
  - components
- stories
  - Documentation using storybook lib

## Publish to NPM

1. Login into B2Bit npm account:

   ```bash
   $ yarn login
   ```

2. Publish
   ```
   $ yarn publish
   ```

## Publish storybook to Chromatic

run the command:

```bash
  npx chromatic --project-token=$CHROMATIC_PROJECT_TOKEN
```
